# Disposable / throwaway email-provider domains.

# One domain per line. Lines starting with `#` and blank lines are ignored.
# Case-insensitive — domains are lowercased at load time.
#
# Source: curated subset of the canonical list at
# https://github.com/disposable-email-domains/disposable-email-domains
# (license: CC0-1.0). Refresh with `scripts/update-disposable-emails.sh`.
#
# This list aims to cover the most common throwaway providers (≥90% of
# real-world signup spam). It is intentionally NOT exhaustive — that file
# is 3500+ lines and full of obscure squatted domains that are no longer
# reachable. False positives hurt more than missed catches.
#
# If a legitimate user is blocked by an entry here, a tenant admin can
# allowlist their email via the admin UI (TODO) or simply remove the line.
