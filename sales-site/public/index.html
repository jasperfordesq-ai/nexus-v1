What is Project NEXUS?

A complete ecosystem for building connected, engaged, and empowered communities — and the infrastructure to connect those communities across borders.

Timebanking Engine

Full credit exchange system with wallet, transactions, and broker controls. Every hour is worth exactly one credit — no exceptions, no hierarchy of value.

Multi-Tenant Platform

Host unlimited communities on a single platform. Each gets its own branding, configuration, and feature set. Parent-child hierarchies with per-tenant feature toggling.

Global Federation

Connect multiple communities into a network for cross-community exchange, shared listings, events, and messaging. Built for a worldwide timebanking economy.

Global Federation:
Connecting Timebanks Worldwide

V1.5 ships with a fully working Federation API — live today, with native protocol adapters for four standards: Nexus, Komunitin (all 15 endpoints), Credit Commons (11 endpoints for CEN-compatible mutual credit), and TimeOverflow. Two-way sync across 9 entity types, event-driven push, reputation portability, and end-to-end integration test coverage. The vision: a global time bank where any platform can discover, connect, and trade time credits across borders.

Why Open Source?

Because transformative technology should be accessible to everyone.

No vendor lock-in, no surprise fees. AGPL-3.0 ensures the platform remains free and that modifications are shared back with the community.

02

Transparent & Auditable

Every line of code is visible. Communities can audit the platform, verify security, and understand exactly how their data is handled.

Community-Driven
